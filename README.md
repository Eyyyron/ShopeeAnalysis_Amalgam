A Data Mining Finals Project – Multi-Year Dataset Curation, Trend Analysis & Forecasting

This repository contains the full workflow, dataset preparation steps, analysis scripts, model development, and documentation for the study “Curating a Multi-Year Dataset for Seasonal Demand Analysis of Uncommon Retail Products on Shopee Philippines.”

The study focuses on uncovering hidden seasonal patterns among uncommon retail items, using multi-year sales data exported from Shopdora. It includes dataset building, time-series decomposition, demand classification, inventory recommendations, and predictive modeling.
